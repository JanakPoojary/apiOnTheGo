Setup your API

Enter the keys you want to add in your api seperated with ',' (coma). Avoid space!

The first feild should be id

For example:

For the following JSON structure:

{
 "data": [
     {
       "id": 1,
       "name": "xyz",
       "email": "xyz@gmail.com"
     }
   ]
}

enter : id,name,email in the setup feild given above.

Thats it! You are all set to go.
